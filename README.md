# Jordan Chen Portfolio

A personal portfolio website for showcasing selected product work, frontend projects, and professional background.

## Tech Stack

- React
- TypeScript
- Vite
- Motion
- Oxlint

Live URL: _Coming soon_

## Local Development

The application lives in the `site` directory.

```bash
cd site
npm install
npm run dev
```

Vite will print the local development URL in the terminal, typically `http://localhost:5173`.

## Build

```bash
cd site
npm install
npm run build
```

The production build is generated in `site/dist`.

## Preview Production Build

```bash
cd site
npm run preview
```
